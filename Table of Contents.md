## Table of Contents

This lab guide provides a structured learning path for Staff and Principal-level DevOps Engineers. It covers cloud infrastructure, infrastructure as code, CI/CD engineering, container platforms, Kubernetes operations, platform stability, observability, security, automation, and developer experience.

**Technologies:** AWS / Terraform / Infrastructure as Code (IaC)

### Topics Covered

  - Network segmentation and connectivity

- 🖥️ **EC2 / RDS / S3 Setup**
